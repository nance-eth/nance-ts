# nance-api

## Examples

basic typescript example:

`ts-node src/api/exmaples/basic.ts`

**GET** 
https://nance-ts-staging.up.railway.app/juicebox

success return
```
{
  "sucess": true,
  "data": {
    "name": "juicebox",
    "currentCycle": "33"
  }
}
```

error return
```
{
  "success":false,
  "error": "space juicebox2 not found!"
}
```
